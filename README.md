# Advisors Listing - Next.js

This project is a small Next.js app that renders a list of advisors with:

- avatar
- name
- price per minute
- availability actions for **Call** and **Chat**

The availability is mocked locally and refreshed automatically every **30 seconds**.

## Tech stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/djdannycastillo/ingenio-advidors-test
cd <project-folder>
```

### 2. Install dependencies

Using npm:

```bash
npm install
```

Or with other package managers:

```bash
yarn
# or
pnpm install
# or
bun install
```

### 3. Run the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

### 4. Open the app

Open your browser and go to:

```bash
http://localhost:3000/advisors
```

> You can start in the home page, there will be a button to go to the advisor list:
>
> http://localhost:3000
